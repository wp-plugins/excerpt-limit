Plugin allows you to edit the length of the excerpt (announcements) posts.

== Description ==

Features:

*   Limit the length of excerpt by the number of words.
*   Limit the length of excerpt on the number of characters.

== Installation ==

1. Unzip files.
2. Upload 'wp-flickr-widget' to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
